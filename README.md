## Resumo: Game Lab (Code.org):


Este repositório contém o resumo estruturado de tudo o que eu aprendi durante as semana 04 a 06 do curso de Ciências da Computação com Game Lab no Code.org.


## Módulo 1: Fundamentos de Desenho (Lições 1 a 4)


### Lição 1: Drawing in Game Lab (Desenho no Game Lab)
 Conheci o ambiente de desenvolvimento "Game Lab" e aprendi a utilizar o sistema de coordenadas ($X$ e $Y$) para posicionar e desenhar minhas primeiras formas geométricas simples na tela.


### Lição 2: Shapes and Parameters (Formas e Parâmetros)
Entendi o papel dos "parâmetros" nas funções de desenho. Aprendi a modificar o tamanho, a largura, a altura e as cores das formas para criar composições visuais mais detalhadas.


### Lição 3: Variables (Variáveis)
Descobri o conceito de variáveis como caixas para armazenar informações (como números ou tamanhos). Aprendi a reutilizar o mesmo valor em vários pontos do meu código, facilitando alterações rápidas.


### Lição 4: Random Numbers (Números Aleatórios)
Aprendi a gerar valores imprevisíveis usando a função de números aleatórios (`random number`), o que me permitiu criar desenhos dinâmicos que mudam de forma a cada vez que o programa é executado.




## Módulo 2: Sprites e Animações (Lições 5 a 7)


### Lição 5: Sprites (Sprites)
Fui introduzida ao conceito de *Sprite* (personagens ou objetos do jogo com uma imagem acoplada). Aprendi a criá-los, defini-los no espaço e desenhá-los na tela do jogo.


### Lição 6: Sprite Properties (Propriedades do Sprite)
Descobri como manipular as características visuais e de posicionamento dos meus sprites (como escala, rotação e coordenadas $X$/$Y$) alterando diretamente as suas propriedades no código.


### Lição 7: Text (Texto)
Aprendi a adicionar e formatar elementos de texto na tela do jogo, controlando tamanho, tipo de fonte e posição, algo essencial para criar placares, instruções ou diálogos.


## Módulo 3: O Loop e Movimento Humano (Lições 8 a 12)


### Lição 8: The Draw Loop (O Loop de Desenho)
Compreendi o funcionamento do *Draw Loop* (um bloco de código que roda repetidamente várias vezes por segundo). Entendi que essa é a base fundamental para criar a ilusão de animação e movimento contínuo.


### Lição 9: Sprite Movement (Movimento do Sprite)
Combinei o uso de variáveis e o *Draw Loop* para atualizar constantemente a posição dos meus sprites, fazendo com que eles ganhassem vida e se movessem sozinhos pela tela de forma fluida.


### Lição 10: Conditionals (Condicionais)
Aprendi sobre lógica booleana e estruturas condicionais (blocos `if` / "se"). Com isso, passei a fazer meu programa tomar decisões próprias e mudar o comportamento do jogo sob certas condições.


### Lição 11: Keyboard Input (Entrada de Teclado)
Descobri como detectar quando uma tecla é pressionada pelo usuário, permitindo que eu controle as ações e a direção dos sprites em tempo real.


### Lição 12: Mouse Input (Entrada de Mouse)
Aprendi a capturar as interações com o mouse (como cliques, movimentos e a posição exata do cursor) para adicionar novas formas de interatividade aos meus projetos.

### Como Executar o Projeto
Para rodar o sistema, certifique-se de que o seu terminal está aberto na pasta raiz do projeto (pdad-explorer-lilian) e execute o comando:
python sistema.py

### Como Instalar as Dependências (Requisitos)
Antes de executar o projeto pela primeira vez, você precisa instalar as bibliotecas necessárias para manipulação de dados, geração de gráficos e leitura de arquivos Excel. Rode o seguinte comando no terminal:
pip install pandas matplotlib openpyxl

### Arquivos de Dados Necessários
Para o sistema funcionar sem erros, os bancos de dados do projeto precisam estar salvos exatamente dentro da pasta dados/ na raiz do projeto, com os seguintes nomes:

moradores.csv (Base de dados dos moradores)
domicilios.xlsx (Base de dados dos domicílios em formato Excel)
